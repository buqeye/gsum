class buqeyemodel.models.MvBLM(dim=None, basis=None, means=None, cov=None, shape=None, scale=None, corr=None)[source]¶

A multivariate Bayesian linear model class

Implements a Bayesian linear model for an Nx1 vector Y:

Y ~ N(Heta, \sigma^2 R)

with an Nxq matrix H, qx1 vector eta, scalar sigma^2, and NxN correlation matrix R. A normal-inverse-gamma prior is placed on eta and sigma^2:

eta, \sigma^2 ~ NIG(means, cov, shape, scale)

The matrix H is generated by a basis function.

class buqeyemodel.models.GeoProcess(dim=None, basis=None, means=None, cov=None, shape=None, scale=None, corr=None)[source]¶

A geometric series with iid random processes as coefficients.

Implements the following model

\[X_k = X_{\mathrm{ref}} \sum_{n=0}^k c_n Q^n\]

where the \(c_n\) are Gaussian processes with parameters using conjugate priors

\[\begin{split}c_n | \beta, \sigma^{2} & \sim N(H \beta, \sigma^2 R) \\ \beta, \sigma^2 & \sim NIG(\mu, V, a, b)\end{split}\]

Conditioning on partial sums \(X_{0}\), \(\dots,\) \(X_k\), allow one to estimate the full summation and obtain posteriors for the parameters.

class buqeyemodel.models.GeoSeries(shape=None, scale=None)[source]¶

A geometric series with iid random variables as coefficients.

Implements the following model

\[X_k = X_{\mathrm{ref}} \sum_{n=0}^k c_n Q^n\]

where the \(c_n\) are iid Gaussians and \(\sigma^2\) has a conjugate prior

\[\begin{split}c_n | \sigma^2 & \sim N(0, \sigma^2) \\ \sigma^2 & \sim IG(a, b)\end{split}\]

Conditioning on partial sums \(X_0\), \(\dots,\) \(X_k\), allow one to estimate the full summation and obtain posteriors for the parameters.
