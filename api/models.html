class buqeyemodel.models.SGP(dim=None, basis=None, means=None, cov=None, shape=None, scale=None, corr=None)[source]¶

A semiparametric Gaussian process class

Treats a function \(y\) as a Gaussian process

\[y(x) | \beta, \sigma^2, \psi \sim N[m(x), \sigma^2 R(x,x; \psi)]\]

with a parameterized mean function \(m\) and correlation function \(R\). The mean at any input point is given by

\[m(x) = h(x)^T \beta\]

where \(h: \mathbb{R}^d \to \mathbb{R}^q\) is a given basis function and \(\beta\) is a \(q\times 1\) vector of random variables. The variance is split into a marginal part \(\sigma^2\) and a correlation function \(R\) that may further depend on parameters \(\psi\), e.g., length scales. A normal-inverse-gamma prior is placed on \(\beta\) and \(\sigma^2\)

\[\beta, \sigma^2 \sim NIG(\mu, V, a, b)\]

The means and covariance of the normal prior placed on \(\beta\) are \(\mu\) and \(\sigma^2 V\), while the shape and scale parameter of the inverse gamma prior placed on \(\sigma^2\) are \(a\) and \(b\).

The dimension of the means vector and the cov matrix, which determines how many mean variables are undetermined in the linear model. Must be greater than 0. By default, dim is inferred by the columns of the basis callable.

basis : callable, optional

The basis function for the mean vector.

evidence(log=True, y=None, **corr_kwargs)[source]¶

Computes the evidence, or marginal likelihood, of the observed data

Specifically, the evidence integrates out \(\beta\) and \(\sigma^2\) such that

\[pr(y | \psi) = \frac{\Gamma(a)}{\Gamma(a_0)} \frac{b_0^{a_0}}{b^a} \sqrt{\frac{|V|}{|V_0|}} [(2\pi)^N |R|]^{-n/2}\]

where subscript 0’s denote prior values. If the priors on \(a_0\) or \(V_0\) are uninformative, then the evidence is undefined, but in this case the evidence is approximated by

\[pr(y | \psi) = \frac{\Gamma(a) \sqrt{|V|}}{b^a} [(2\pi)^N |R|]^{-n/2}\]

This is appropriate for model comparison since the factor due to priors is only a constant and hence cancels.

Parameters:

log : bool, optional

Whether to return the log of the evidence, which can be useful for numerical reasons.

y : (n, N) array, optional

Data for which to compute the evidence. Defaults to the data passed to the observe method.

**corr_kwargs : optional

Keyword arguments passed to the correlation function. Defaults to those passed to the observe method.
Returns:

scalar

The (log) evidence

\[pr(\ell | y, ...) \propto pr(y | \ell, ...) pr(\ell)\]

for any correlation parameter \(\ell\). The evidence given \(\ell\) and the other correlation parameters (…) is then used to compute the posterior.

student_params(X=None, H=None, R=None, y=None, **corr_kwargs)[source]¶

Returns the parameters of the student \(t\) distribution.

Given a function

\[y | \beta, \sigma^2, \psi \sim N[H\beta, \sigma^2 R]\]

with a normal inverse gamma prior on \(\beta, \sigma^2\),

\[\beta, \sigma^2 \sim NIG(\mu, V, a, b)\]

the integrated process is given by

\[y | \psi \sim MVT\left[2a, H\mu, \frac{b}{a} (R + HVH^T)\right]\]

If data has been observed, then posterior values for \(\mu,V,a,b\) are used.

Parameters:

X : (N, d) array, optional

The input points at which to compute the basis H and R. If None, then the defaults from the observe method are used.

H : (N, q) array, optional

The basis function evaluated at \(N\) points. If None, then the basis function is computed at X.

R : (N, N) array, optional

The correlation matrix. If None, then the correlation function is computed using X and **corr_kwargs.

y : (n, N) array, optional

Observed GP values used to compute the updated normal-inverse-gamma hyperparameters. If None, then the data passed to the observe method are used. If observe has not been called, prior values are used.

**corr_kwargs : optional

Optional keyword arguments for the correlation function. If none are provided, defaults from the observe method are used instead.
Returns:

tuple

The degrees of freedom, mean, and sigma matrix of a multivariate \(t\) distribution.
class buqeyemodel.models.PowerProcess(dim=None, basis=None, means=None, cov=None, shape=None, scale=None, corr=None)[source]¶

A power series with iid random processes as coefficients.

Implements the following model

\[S_k(x) = S_{\mathrm{ref}}(x) \sum_{n=0}^k c_n(x) r(x)^n\]

where the \(c_n\) are Gaussian processes with parameters using conjugate priors

\[\begin{split}c_n(x) | \beta, \sigma^2, \psi & \sim N(m(x), \sigma^2 R(x,x;\psi)) \\ \beta, \sigma^2 & \sim NIG(\mu, V, a, b)\end{split}\]

Conditioning on partial sums \(S_{0}\), \(\dots,\) \(S_k\), allow one to estimate the full summation and obtain posteriors for the parameters.

Computes the evidence, or marginal likelihood, of the partial sums

Specifically, the evidence integrates out \(\beta\) and \(\sigma^2\) such that

\[\begin{split}pr({S_i} | \psi, r, S_{\mathrm{ref}}) & = \frac{pr(y | \psi)}{\prod_i |S_{\mathrm{ref}} r^i|} \\ & = \frac{1}{\prod_n |S_{\mathrm{ref}} r^n|} \frac{\Gamma(a)}{\Gamma(a_0)} \frac{b_0^{a_0}}{b^a} \sqrt{\frac{|V|}{|V_0|}} [(2\pi)^N |R|]^{-n/2}\end{split}\]

This is appropriate for model comparison since the factor due to priors is only a constant and hence cancels.

observe(X, partials, ratio, ref=1, orders=None, leading_kwargs=None, ratio_kwargs=None, **corr_kwargs)[source]¶

Observe the partial sums of the series and update parameters.

The partial sums are observed at the input locations X. Using the given ratio and ref, the coefficients of the series are extracted and are used to create posterior distributions for the mean and variance parameters. One can then interpolate partial sums or predict the value of the full summation at new points. Additional functionality, including evidence calculations and posteriors for the ratio and correlation functions are possible.

Parameters:

X : (N,d) array

The :math`N` input locations where the partial sums are observed. Columns correspond to the dimensionality of the input space. If 1D, X must be an \(N \times 1\) column vector.

partials : (n,N) array

The \(n\) lowest known partial sums, each with \(N\) points observed along each curve.

ratio : callable, scalar, or (N,) array

The value of the ratio that scales each order in the power series with increasing powers. If callable, it must accept X as its first argument and can optionally accept **ratio_kwargs.

ref : callable, scalar, or length (N,) array

The overall scale of the power series. If callable, it must accept X as its first argument. The default value is 1.

orders : (n,) array

The orders of the given partial sums. If None, it is assumed that all orders from 0 to math:n are given: [0, 1, ..., n].

leading_kwargs : dict

Keyword arguments passed to an SGP initializer for the leading order coefficient. This allows the leading order to be treated differently than the other orders of the series. corr_kwargs can also be in this dict and will be passed to the observe method of the leading order SGP. Defaults to None, in which case the leading order is an iid coefficient like the others.

ratio_kwargs : dict

Additional keyword arguments passed to the ratio function. Defaults to None.

**corr_kwargs : optional

Additional keyword arguments passed to the correlation function. These values will be saved and used as defaults for all other methods.

class buqeyemodel.models.PowerSeries(shape=None, scale=None)[source]¶

A power series with iid random variables as coefficients.

Implements the following model

\[S_k = S_{\mathrm{ref}} \sum_{n=0}^k c_n r^n\]

where the \(c_n\) are iid Gaussians and \(\sigma^2\) has a conjugate prior

\[\begin{split}c_n | \sigma^2 & \sim N(0, \sigma^2) \\ \sigma^2 & \sim IG(a, b)\end{split}\]

Conditioning on partial sums \(S_0\), \(\dots,\) \(S_k\), allow one to estimate the full summation and obtain posteriors for the parameters.
